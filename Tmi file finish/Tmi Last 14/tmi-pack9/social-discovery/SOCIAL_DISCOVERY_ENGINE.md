# DISCOVERY_ENGINE.md
## How Artists Get Found

---

## PURPOSE
TMI is discovery-first. Higher rank number = less established = shown FIRST.
This is the core discovery rule and must be enforced everywhere.

---

## DISCOVERY SURFACES

| Surface | Sort Rule | Location |
|---|---|---|
| Preview Stack Wall | Rank# descending (least known first) | Homepage Page 3 |
| Magazine Page 2 | Featured + Rank blend | Homepage Page 2 |
| Search results | Relevance + Rank blend | Search |
| Billboard Board | Points and momentum | Billboard page |
| Artist spotlight | Algorithm rotation | Side panels |
| "You might like" | Fan listening history | Profile sidebar |

---

## DISCOVERY ALGORITHM INPUTS

The algorithm uses:
- Artist rank# (discovery order)
- Momentum score (recent activity)
- Genre tags
- Fan listening history
- Geographic signals (optional)
- Time of day (some genres peak at different times)
- Recent event activity
- Streak data

---

## DISCOVERY ALGORITHM GUARDRAILS

The algorithm CANNOT:
- Remove any artist from discovery entirely
- Permanently suppress an artist
- Use paid placement without disclosure
- Show the same artist twice in a 10-card row

# SEARCH_ENGINE.md
## How Every Search Works

---

## SEARCH SCOPE

Users can search across:
- Artists
- Events / shows
- Articles
- Rooms / venues
- Song/album titles (if tagged)
- Genre
- Host names

---

## SEARCH RESULT CARD

Each result shows:
- Avatar / thumbnail
- Name
- Type label (Artist / Article / Show / Room)
- One-line description
- Tier badge (if applicable)
- Quick action (Follow / Join / Read)

---

## SEARCH RANKING

Results ranked by:
1. Exact match (name/title)
2. Partial match
3. Tag match
4. Momentum score
5. Discovery rank blend (surfaces newer artists)

---

## SEARCH BOT

A lightweight search bot assists:
- Autocomplete suggestions
- Did-you-mean corrections
- "Nothing found" fallback: suggests nearby/related
- Filters: genre, tier, live now, upcoming

# FOLLOW_SYSTEM.md
## Artist Follows and Fan Relationships

---

## HOW FOLLOWING WORKS

Fan clicks Follow on artist profile:
1. Artist added to fan's Following list
2. Fan added to artist's Followers list
3. Fan receives notifications for artist events
4. Fan's discovery feed weights artist higher
5. Fan earns +5 points for first follow of any artist

---

## WHAT FANS SEE (Following Feed)

After following artists, fan has a "Following" tab:
- Recent events from followed artists
- New articles from followed artists
- Upcoming shows from followed artists
- "Going Live" notifications from followed artists

---

## WHAT ARTISTS SEE (Followers View)

Artists can see:
- Total follower count
- New followers today
- Follower tier breakdown (Free/Bronze/Gold/Diamond)
- Top fans (by tips + reactions)
- Follower growth graph

# FAN_CLUBS.md
## Fan Groups and Community Around Artists

---

## CONCEPT
Every artist can have an official Fan Club.
It's a private community space within TMI for their most engaged fans.

---

## FAN CLUB TIERS

| Club Level | Requirement | Benefits |
|---|---|---|
| Open | Follow artist | Basic club access |
| Member | 500 pts + follow | Circle discussions |
| Inner Circle | 5,000 pts + follow | Priority seating at events |
| VIP Member | Gold subscription + follow | Backstage access, artist chat |
| Founding Member | Artist invites | Lifetime badge + perks |

---

## FAN CLUB FEATURES

- Discussion circle (The Circle room format, fan-only)
- Member directory
- Club achievements and challenges
- Exclusive polls
- Artist posts (text/audio/video) for club only
- Event early access

# SHARED_ROOMS.md
## Watching Events Together as a Group

---

## CONCEPT
Fans can create a private viewing group.
They watch the same event in the same venue together, in adjacent seats.

---

## HOW IT WORKS

1. Fan creates a "Watch Party" for a specific event
2. Gets shareable invite link
3. Friends join via link
4. System reserves adjacent seats for the group
5. Group can see each other's reactions
6. Private group chat overlay (in addition to public)
7. Group has shared energy meter contribution

---

## GROUP SIZE LIMITS

| User Tier | Max Group Size |
|---|---|
| Free | 4 |
| Bronze | 8 |
| Gold | 16 |
| Diamond | 32 |
| Signature | 50 |
