# ISSUE_ARCHIVE_SYSTEM.md
## How Weekly Issues Are Created and Stored

---

## CONCEPT
Every week is a new issue of The Musician's Index.
Issues have: a theme, a crown winner, featured articles, cyphers, playlists.
Past issues are archived like a real magazine back-catalog.

---

## ISSUE STRUCTURE

Each issue contains:
```json
{
  "issue_id": "TMI-2026-W12",
  "issue_number": 12,
  "week_of": "2026-03-16",
  "theme": "Underground Rises",
  "crown_winner": { "artist_id": "...", "artist_name": "...", "weeks_held": 3 },
  "featured_articles": [...],
  "top_10_chart": [...],
  "weekly_playlist": [...],
  "cypher_recap": {...},
  "premiere_events": [...],
  "sponsor_campaigns": [...],
  "crown_history": [...],
  "top_hype_moment": { "timestamp": ..., "clip_url": "..." }
}
```

---

## ARCHIVIST BOT

Runs every Sunday at midnight (same as Crown Bot):

```
1. Collect week's content (articles, clips, crown data)
2. Compile issue JSON
3. Take "snapshot" of homepage layout
4. Generate "Past Issue" PDF/flipbook (optional)
5. Save to /archive/issues/
6. Update "Past Issues" library page
7. Update Homepage 1's "Previous Issue" section
8. Log attendance: who watched what this week
```

# STREAMWIN_PLAYLIST_PIPELINE.md
## Streaming Music for Points — Full System

---

## CONCEPT
Stream & Win allows fans to earn points by streaming music playlists.
Each playlist is curated by issue/genre/artist.
Points accumulate based on actual listening time.

---

## HOW IT WORKS

```
Fan opens Stream & Win (any page widget or dedicated page)
    ↓
Selects playlist (issue theme, genre, curated)
    ↓
Music streams
    ↓
Per 5 minutes of streaming:
    +10 points
    ↓
Daily cap: 60 points from streaming alone
    ↓
Bonus: +20 points if it's a featured playlist for current issue
    ↓
Points display updates in real time on widget
```

---

## PLAYLIST TYPES

| Type | Source | Points Multiplier |
|---|---|---|
| Issue Playlist | Weekly curated | 1.2× |
| Genre Playlist | Auto-generated by genre | 1.0× |
| Artist Playlist | Artist uploads | 1.0× |
| Crown Winner Playlist | Automatic from #1 | 1.5× |
| Discovery Playlist | Undiscovered artists only | 1.3× |
| Sponsored Playlist | Brand-supported | 1.0× |

---

## ANTI-FRAUD RULES

```
MINIMUM stream threshold: 30 seconds per track counts
Tab must be active (not background-tabbed)
IP-based daily point cap
Same track replayed: points only count first 3 plays per day
Auto-muted streams: do not count
```

---

## PROFILE LIBRARY MODULE

When fan saves a playlist:
- Appears in their Library on profile
- Can be played from profile page
- Shows artist attribution
- Privacy: Public, Friends, or Private

---

## ARTIST PROFILE PLAYLIST MODULE

Artists can pin featured playlists to their profile:
- "My Latest Playlist"
- "For My Fans"
- "Issue Feature Playlist"

## PURPOSE

## HOW IT WORKS

```
User logs in → "Daily Spin" card appears on dashboard
```

# ARTIST_DISCOVERY_ALGORITHM.md
## How the Platform Surfaces Undiscovered Artists

---

## CORE PRINCIPLE

The Musician's Index is discovery-first.
Higher rank number = less established = shown FIRST.
The algorithm actively fights against popularity bias.

---

## DISCOVERY SURFACES

| Surface | Sort Rule |
|---|---|
| Homepage 1 (Cover) | Rank# descending in outer ring positions |
| Homepage 2 Lobby Wall | Least viewers first |
| Homepage 2 Undiscovered Boost | Lowest total streams artist |
| Homepage 3 Top 10 Charts | Weighted blend of activity + discovery |
| Homepage 3 Genre Hexagon → Enter | Discovery order on genre page |
| Preview Lobby list | Viewer count ascending |
| Search default | Relevance + discovery blend |

---

## ALGORITHM INPUTS

```
discovery_score = (
  (reverse_rank * 0.3) +          // Higher rank# = more surface
  (freshness_score * 0.25) +      // Recent activity boost
  (session_engagement * 0.2) +    // Quality of recent streams
  (genre_diversity * 0.15) +      // Fills underserved genre slots
  (new_account_bonus * 0.1)       // First 30 days extra boost
)
```

---

## GUARDRAILS

The algorithm CANNOT:
- Permanently bury any artist
- Give paid placement without disclosure
- Show the same artist twice in any 10-card row
- Suppress artists based on genre (all genres get representation)
