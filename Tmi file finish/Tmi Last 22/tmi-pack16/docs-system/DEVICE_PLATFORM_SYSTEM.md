# DEVICE_PLATFORM_SYSTEM.md
## Every Screen — Device Types and Render Modes
Desktop/Laptop (browser): Full Mode — all panels, sidebar, HUD strip, operator overlay
Mobile Phone (iOS/Android): Compact Mode — vertical scroll, swipe pages, bottom nav, compact HUD
Tablet (iPad/Android): Hybrid Mode — sidebar optional, larger room tiles, compact operator
Smart TV (Apple/Android/Roku): Watch Mode — main stage 80%, D-pad navigation, large text
Venue Screen (kiosk): Display Mode — lobby wall or scoreboard, no interaction, auto-rotate
Operator Station: Operator Mode — full command center layout, all health panels
Second Screen: Phone becomes remote → TV shows main stage, phone shows chat/reactions/tips
App Distribution: Free download everywhere. Subscription inside. Never hard-lock on lapse.
Phase 1: Web PWA + iOS + Android
Phase 2: Apple TV + Android TV
Phase 3: Roku + Samsung + LG + Desktop apps
