# OVERLAY_STACKING_SYSTEM.md
## What Appears Above What — 10-Layer Visual Priority
Layer 1: Page/Belt content (homepage, magazine, artist profile)
Layer 2: Room surface (stage, queue, roster, chat)
Layer 3: Shared Preview Window (docked to side — never over performer face)
Layer 4: Timer/Countdown overlays (battle round, event start)
Layer 5: Announcement cards (winner, next up, prize)
Layer 6: Sponsor/Ad overlays (max 10s animated, then static, clearly labeled)
Layer 7: HUD strip (bottom bar — audio, points, notifications)
Layer 8: Notification popups (brief, auto-dismiss)
Layer 9: Emergency broadcasts (Big Ace only, full-width, all others pause)
Layer 10: System alerts (moderation, safety — highest visible)
Preview window: never covers artist face, stage center, or turn timer.
Sponsor overlays: banned at winner reveal, crown transfer, tier upgrade.
