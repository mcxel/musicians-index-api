## APPLE APP STORE CHECKLIST

### Technical Requirements
- [ ] Xcode project configured with correct Bundle ID: `com.berntoutglobal.themusiciansindex`
- [ ] Minimum iOS version: 15.0
- [ ] App built for all required architectures (arm64)
- [ ] All frameworks properly linked
- [ ] No private API usage
- [ ] App does not crash on launch
- [ ] App works without network connection (offline state handled)

### Store Listing Requirements
- [ ] App name: "The Musician's Index" (exactly)
- [ ] Bundle ID: `com.berntoutglobal.themusiciansindex`
- [ ] Privacy policy URL: `https://themusiciansindex.com/privacy`
- [ ] Terms of service URL: `https://themusiciansindex.com/terms`
- [ ] Support URL: `https://themusiciansindex.com/support`
- [ ] App description: Written, spell-checked, "Berntout" verified
- [ ] Age rating completed: Likely 12+ (music platform, mild language possible)
- [ ] Keywords: 100 char max, relevant music keywords

### Visual Assets
- [ ] App icon: 1024×1024 PNG, no alpha channel, no transparency
- [ ] Screenshots: 6.7" (required), 12.9" iPad (required), 5.5" (recommended)
- [ ] App preview video: 15-30 seconds (optional but recommended)

### In-App Purchases
- [ ] All subscription tiers created in App Store Connect
- [ ] Bronze $4.99/mo, Gold $9.99/mo, Diamond $19.99/mo, Artist Pro $29.99/mo
- [ ] "Restore Purchases" button implemented and tested
- [ ] App Store Server Notifications configured in Render API

### Privacy
- [ ] Privacy nutrition labels completed accurately
- [ ] Sign in with Apple implemented (required if any social login offered)
- [ ] App Tracking Transparency (ATT) prompt if any tracking
