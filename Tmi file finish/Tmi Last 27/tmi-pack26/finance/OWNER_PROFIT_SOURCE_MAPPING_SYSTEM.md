## OWNER PROFIT FLOW BY SOURCE

### Tips
```
Fan pays $10 tip
  Stripe fee: ~$0.59
  Artist receives: $7.00 (70%)
  Platform net: $2.41 (30% - fee)
  Flows to: platform net revenue → weekly distribution
```

### Beat License (Basic $25)
```
Artist pays $25 for basic beat license
  Stripe fee: ~$1.03
  Producer receives: $20.00 (80%)
  Platform net: $3.97 (20% - fee)
  Flows to: platform net revenue → weekly distribution
```

### Ticket Sale ($9.99 — max price)
```
Fan pays $9.99 for ticket
  Stripe fee: ~$0.59
  Event host receives: $7.49 (75%)
  Platform net: $1.91 (25% - fee)
  Flows to: platform net revenue → weekly distribution
```

### Fan Club Subscription ($9.99/mo VIP)
```
Fan pays $9.99/month
  Stripe fee: ~$0.59
  Artist receives: $6.99 (70%)
  Platform net: $2.41 (30% - fee)
  Flows to: platform net revenue → weekly distribution
```

### Fan Credits ($5 for 100 credits)
```
Fan pays $5.00
  Stripe fee: ~$0.45
  Artist receives: $0 (credits are platform currency)
  Platform net: $4.55 (100% - fee)
  Flows to: platform net revenue → weekly distribution
```

### Avatar Store ($X per item)
```
Fan pays item price
  Stripe fee: ~3%
  Creator receives: $0 (platform-owned items)
  Platform net: price - fee
  Flows to: platform net revenue → weekly distribution
```

### Sponsor Packages ($99–$999+/month)
```
Sponsor pays invoice
  No Stripe fee (invoice-based)
  Platform net: 100%
  Flows to: platform net revenue → weekly distribution
  Note: Sponsor invoices billed separately, not through artist split
```

## REVENUE THAT DOES NOT DIRECTLY FLOW TO OWNERS

```
Artist payout withdrawals: money leaving platform to artist (already split)
Stripe platform reserve: Stripe holds this — not owner money
Refunds: reduce gross revenue (already deducted before distribution)
Fan credits used: platform liability until spent by fan
```

## HOW TO MAXIMIZE PLATFORM NET PROFIT OVER TIME

```
Growth levers (more gross revenue):
  + More active artists = more rooms = more tips
  + More producers = more beats = more licenses
  + More events = more tickets
  + More fan club members = recurring subscription revenue
  + More sponsors = invoice revenue

Efficiency levers (more of gross becomes net):
  + Lower refund rate (better dispute resolution)
  + Lower chargeback rate (better anti-bot protections)
  + Better Stripe pricing (negotiate after 6 months of volume)
  + Lower operating costs as platform scales (Render volume discounts)
```
