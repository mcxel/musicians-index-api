# Universe Principles

The Musician's Index is a living music magazine universe.

## Core Rule
No isolated features are allowed. Every feature must belong to:
- a route
- a scene
- a script
- an engine
- a bot chain
- a fallback state
- a recap/archive path
- a validation path

## Implications
- All features must connect through the system architecture
- No standalone widgets or pages
- Every experience has fallback handling
- All content has recap/archive paths
