# Magazine Flow Structure

This document outlines the core flow structure of the TMI magazine.

## The Magazine Spine

The permanent issue flow, or "Magazine Spine," is:

**Artist Page → Article/News Page → Random Page → repeat**

This sequence is the backbone of the magazine experience.

-   **Artist pages** are the entry anchors for discovery.
-   **Article pages** provide the editorial narrative that gives context and depth to the artists.
-   **Random pages** provide rhythm, monetization opportunities, and surprise to keep the experience engaging.

The `SequenceController` system is responsible for enforcing this pattern.
