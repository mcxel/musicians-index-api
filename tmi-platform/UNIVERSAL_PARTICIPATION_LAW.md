## VIII. POINT CATEGORIES (SEPARATED)

**CRITICAL: These must be kept separate to prevent inflation.**

### Engagement Points
- Used for rank calculation
- Non-spendable
- Public display
- Total XP equivalent

### Reward Points
- Spendable on store
- Limited daily earn cap
- Convertible to cosmetics, skins, emotes
- Transferable (limited)

### Entry Points
- Used for entering competitions
- Event-specific
- Non-transferable
- Capped per event

### Achievement Points
- Earned on unlock only
- Used for achievement level
- Non-spendable
- Display only

### XP (Level Progression)
- Role-specific (Fan/Artist/Producer/etc)
- Separate progression per role
- Used for leveling
- Determines access/perks

## XI. ANTI-FARM RULES (Protection)

Required rules to protect the economy:

```
✗ AFK Detection
  - Must interact every 5 minutes to earn presence points
  - Passive watching after 5min = zero points until next interaction
  
✗ Spam Detection
  - Max 10 same-type actions per minute
  - Repeated identical actions flagged
  
✗ Duplicate Answer Detection
  - One answer per question per user per event
  - Repeat answers flagged
  
✗ Bot Abuse Detection
  - User account activity signatures checked
  - Unusual patterns flagged
  - IP rotation detected
  
✗ Idle Penalties
  - Earning stops after 20 min inactivity
  - Resume earning on next interaction
  
✗ Collusion Detection
  - Vote patterns analyzed for coordination
  - Same voters on same contestants flagged
  
✗ Vote Ring Detection
  - User clustering analyzed
  - >5 votes from same source in 10 sec = flagged
  
✗ Fake Attendance Detection
  - Video stream verification (future)
  - Microphone/camera presence checks
  
✗ Fast-Click Detection
  - Superhuman click rates flagged
  - Impossible answer speed detected
```

## XIII. CRITICAL RULES

1. **Participation is the only universal currency.**
   - Winning is bonus, not requirement.
   - Showing up is earning.

2. **Everyone starts equal.**
   - No purchased advantages.
   - Only earned progression.

3. **Multipliers are earned, not bought.**
   - Streaks must be genuine.
   - Event bonuses are temporary.

4. **Points are auditable.**
   - Every point has a source.
   - Every transaction logged.
   - Fraud = immediate reversal.

5. **Separate point buckets prevent inflation.**
   - Engagement ≠ Reward ≠ Entry ≠ Achievement ≠ XP
   - Each has own earning cap
   - Each has own redemption rules
