Accessibility Statement

Last Updated: February 17, 2026

Our Commitment

The Musician's Index is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

Accessibility Features

The Musician's Index includes the following accessibility features:

  • Keyboard Navigation: Full site navigation using keyboard only (Tab, Enter, Arrow keys)
  • Screen Reader Support: Compatible with JAWS, NVDA, VoiceOver, and TalkBack
  • Alternative Text: All images include descriptive alt text
  • High Contrast: Sufficient color contrast ratios (minimum 4.5:1 for text)
  • Resizable Text: Text can be resized up to 200% without loss of functionality
  • Focus Indicators: Visible focus indicators for all interactive elements
  • Captions: Video content includes captions and transcripts
  • ARIA Labels: Proper ARIA landmarks and labels for screen readers

Supported Technologies

This website is designed to work with the following assistive technologies:

  • Screen readers (JAWS 2023+, NVDA 2023+, VoiceOver, TalkBack)
  • Screen magnification software (ZoomText, MAGic)
  • Speech recognition software (Dragon NaturallySpeaking)
  • Alternative input devices (sip-and-puff, head pointer, eye tracking)

Known Limitations

Despite our efforts, some areas may not yet be fully accessible:

  • Some third-party embedded content may not meet accessibility standards
  • Older PDF documents may not be screen reader compatible (we're working to update them)
  • Complex data visualizations may require alternative text descriptions

We are actively working to address these issues.

Ongoing Efforts

We are continuously improving accessibility through:

  • Regular accessibility audits and testing
  • Staff training on accessibility best practices
  • User testing with people who use assistive technologies
  • Automated accessibility scanning on every code deployment
  • Adherence to WCAG 2.1 Level AA during design and development

Standards & Guidelines

The Musician's Index strives to comply with:

  • WCAG 2.1 Level AA (Web Content Accessibility Guidelines)
  • Section 508 (U.S. Federal accessibility standards)
  • ADA (Americans with Disabilities Act)
  • EN 301 549 (European accessibility standard)

Accessible Content Tips

To get the most accessible experience on our site:

  • Use the latest version of your browser
  • Enable JavaScript for full functionality
  • Adjust text size in your browser settings (Ctrl/Cmd + Plus/Minus)
  • Use high contrast mode if available in your OS
