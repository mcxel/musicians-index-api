## SECONDARY EVENT ROTATION

All hosts except Monthly Idol anchors can rotate into:
- Singer vs Singer battles
- Drummer vs Drummer battles
- Dancer vs Dancer battles
- Comedian vs Comedian battles
- Pianist vs Pianist battles
- Cyphers
- Dirty Dozen

**Anti-repetition rule**: Same host cannot appear twice in a row in rotation events.

**Monthly Idol panel rule**: Meridicus, Aiko, and Zahra remain SEATED anchors in Monthly Idol mode, but may roam in rotation events.

## VISUAL STYLE CANON

All characters must:
- Exist in neon-lit stage environments (orange, cyan, purple glow)
- Use soft cinematic lighting
- Feature gold accents
- Have slightly stylized proportions (not fully realistic, not fully cartoon)
- Match Bobby Stanley's cartoony-premium art style

---

## WORLD OBJECT PHILOSOPHY

All props and environment items must be **hybrid retro-futuristic**:
- Real-world object base (paper, metal, wood, glass)
- Futuristic augmentation (neon edges, holographic overlays, glowing UI)
- Believable physical behavior + digital enhancement
