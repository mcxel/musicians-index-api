# HOST_CONVERSATION_LOOP.md
## The Interactive Back-and-Forth System

---

## LOOP STRUCTURE

```
1. HOST: Delivers prompt (question/challenge)
2. CONTESTANT: Responds
3. HOST: Receives response
4. HOST: Interprets content + tone
5. HOST: Selects response category
6. HOST: Delivers in-character reply
7. [OPTIONAL] HOST: Follow-up question
8. [OPTIONAL] CONTESTANT: Second response
9. HOST: Concludes or hands off to gameplay
```

## MAX DEPTH BY EVENT TYPE
| Event | Max Exchanges |
|---|---|
| Deal or Feud | 2–3 |
| Circles & Squares | 3–4 |
| Monthly Idol | 4–6 |
| Monday Night Stage | 2–3 |
| Battles / Cyphers | 1–3 |
| Dirty Dozen | 2–4 |
