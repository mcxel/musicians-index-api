# HOST_FOLLOWUP_RULES.md
## When and How Hosts Ask Follow-Up Questions

---

## FOLLOW UP WHEN
- Answer was unclear or incomplete
- Emotional depth surfaced
- Show momentum benefits from more
- Crowd visibly reacted

## DO NOT FOLLOW UP WHEN
- Event pace requires moving on
- Max conversation depth reached
- Contestant clearly finished
- Answer was definitive

## FOLLOW-UP TYPES
| Type | Example |
|---|---|
| Clarification | "Can you say more about that?" |
| Challenge | "But here's the thing..." |
| Encouragement | "Go deeper. I think there's more." |
| Funny | "That's it? That's all you've got?" |
| Reveal | "Let's see if the audience agrees." |

# HOST_LISTENING_ENGINE.md
## How Hosts Behave While Contestant Speaks

---

## LISTENING STATES
| State | Animation | Audio |
|---|---|---|
| Active listen | Head tilt, focused gaze | Silence or crowd |
| Nodding | Slow head bob | Soft affirmation sound |
| Intrigued | Lean forward, brow raise | None |
| Surprised | Quick react, eye widen | Brief crowd stir |
| Ready to respond | Subtle mic raise | None |

## INTERRUPTION RULES
- Cannot interrupt until contestant has spoken minimum 3 seconds
- Can react with expressions mid-speech
- Can nod or laugh briefly without cutting off
- Must wait for natural pause before responding

## HANDOFF
After contestant speaks:
1. Brief hold (0.5–1 second)
2. Reaction expression
3. Begin response

This natural beat makes the conversation feel real.
