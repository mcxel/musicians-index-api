# HOST_MOBILITY_RULES.md
## Host Movement Classification and Rules

---

## PURPOSE
Defines how each host moves through their stage environment.
Movement must match the host's personality and show type.

---

## MOBILITY CLASSES

### Class A: Free-Roam
Full stage movement permitted.

**Members**: Bobby Stanley, Timothy Hadley, Nova Blaze, VEX, Julius

**Allowed Movement**:
- Walk across full stage
- Approach contestants
- Cross between zones
- Inspect props
- Greet audience sections
- Move toward battle circles
- Roam sponsor areas
- Shift between camera positions
- React physically to moments

**Prohibited**: Entering seat panel area of Monthly Idol without specific scene event

---

### Class B: Anchored + Hybrid
Primarily seat-locked but capable of expressive movement.

**Members**: Meridicus James, Aiko Starling, Zahra Voss (in Monthly Idol)
**When Roaming**: These hosts become Class A in rotation/battle events

**Anchored Mode Allowed Movement**:
- Lean forward toward contestant
- Lean sideways toward co-host
- Turn in chair toward stage
- Point across table
- Stand up briefly (max 3 seconds before reseat for minor reaction)
- Full stand for winner reveal
- Full stand for surprise/shock reaction
- Return to seated position cleanly
- Clap while seated
- Strong react (recoil, lean forward fast)
- Subtle chair sway

**Anchored Mode Prohibited**:
- Walking away from table mid-judging
- Turning back to crowd without reason
- Standing during another judge's speak

**Hybrid Mode (rotation events)**:
- All Class A movement permitted
- Home seat point becomes performer zone center

---

### Class C: Stage Patrol
VEX-specific movement logic.

**Members**: VEX

**Movement Rules**:
- Enters from stage wings or defined VEX lane
- Approaches contestant in proximity-escalating sequence
- Has defined escort path per stage layout
- Exit via stage wings
- Can deviate slightly for comedic choreography
- Must not block scoreboard, camera, or judge panel

## SEATED HOST PERFORMANCE SYSTEM

Monthly Idol panel hosts must feel alive while seated.

**Seated Animation Library** (required):
- Chair idle (breathing, subtle sway)
- Forward lean (interest/reaction)
- Side lean to co-host (whisper/banter moment)
- Turn in chair (facing different direction)
- Hand-to-chin (deliberation)
- Note-taking (pen on pad motion)
- Clap seated (enthusiastic)
- Table tap (emphasis)
- Quick half-stand
- Full stand for winner reveal
- Strong recoil (surprise)
- Proud lean-back (satisfaction)
- Empathetic lean-in (emotional contestant moment)
- Deliberation huddle with co-host

**Return-to-seat rule**: After any stand, host reseats within 4-6 seconds unless event holds them standing.

---

## HOST IDLE SUPPRESSION RULES

When another cast member is speaking:
- Background hosts show listening pose only
- No micro-animations that distract
- No prop interaction
- Seated anchors: still, facing speaker
- Julius: reduced size / moved to edge / minimal motion
- VEX: stage wings only (not on stage unless called)
