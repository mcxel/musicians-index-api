# HOST_RESPONSE_INTELLIGENCE.md
## How Hosts Read and React to Contestant Answers

---

## SIGNAL DETECTION

| Contestant Signal | Host Behavior |
|---|---|
| Confidence | Match energy, affirm, potentially challenge |
| Hesitation | Reassure, give space, support |
| Confusion | Clarify, rephrase, guide |
| Humor | Lean in, match the moment |
| Excitement | Amplify, hype the crowd |
| Disappointment | Recover gracefully, find the positive |
| Overconfidence | Light playful challenge |
| Nervousness | Warm support or redirect |
| Strong answer | Praise + advance the show |
| Weak answer | Recover + transition gracefully |

---

## EMOTIONAL TONE READ

Hosts detect:
- Volume/energy level of response
- Keyword sentiment (positive/negative/neutral)
- Response length (short = brief, long = engaged)
- Confidence markers in phrasing

---

## RESPONSE CATEGORIES

Based on signal, host selects from:
1. Praise
2. Challenge
3. Support
4. Clarify
5. Hype
6. Recover
7. Tease
8. Reveal
9. Transition
