# WARDROBE_ENGINE.md
## Universal Outfit System for All Cast Members

---

## PURPOSE
Every cast member changes outfits. Outfits must match the show, venue, energy, and season.
Outfits are also shoppable — this is a monetization layer.

---

## UNIVERSAL OUTFIT SLOTS

| Slot | Description | When Used |
|---|---|---|
| `default` | Signature show look | Primary show |
| `premium` | Upgraded event version | High-tier events |
| `battle` | Battle/Cypher energy | Rotation events |
| `idol` | Monthly Idol specific | Idol only |
| `sponsor` | Brand partnership fit | Sponsor events |
| `seasonal` | Holiday/seasonal themed | Calendar events |
| `casual` | Low-key mode | Idle/light events |
| `fallback` | Sprite-safe lightweight | Low-power devices |

---

## ROTATION RULES

1. No outfit repeated back-to-back
2. No outfit repeated within same session
3. Event type influences category selection
4. Sponsor outfits prioritized on sponsor events
5. Seasonal outfits auto-activate on calendar triggers
6. VEX has strictest rule: never repeats at all in same show

# STYLE_RULES.md
## Visual Style Laws for All Cast Members

---

## CORE VISUAL CANON

All cast members must exist in the same world:
- Neon-lit stage environments
- Soft cinematic lighting on skin
- Gold accents as signature markers
- Slightly stylized proportions (Bobby Stanley cartoony level)
- Premium, polished, performance-ready

---

## WORLD OBJECT PHILOSOPHY

All props and items must be **retro-real + futuristic**:

| Layer | Description |
|---|---|
| Base | Real-world material (paper, metal, wood, fabric, leather) |
| Augment | Futuristic glow (neon edges, holographic overlay, reactive lighting) |
| Behavior | Interactive (opens, flips, projects, responds to events) |

**Example: Scoreboard**
- Base: mechanical flip-board
- Augment: digital number overlay, glow accent
- Behavior: physically flips AND digitally updates

---

## HYBRID PROP EXAMPLES

| Prop | Real Base | Futuristic Upgrade |
|---|---|---|
| Microphone | Vintage stage mic | Neon ring + sound pulse |
| VIP Ticket | Paper, perforated edge | Glowing barcode + shimmer |
| Magic Hat | Felt magician hat | Glowing interior + particle smoke |
| Scoreboard | Mechanical flipboard | Digital overlay + glow |
| Flashlight | Metal casing | Scanning beam UI |
| Monitor | Old screen/tablet | Holographic expansion |

---

## VISUAL RULES (NON-NEGOTIABLE)

1. No generic flat colors — always glow, depth, or texture
2. No white default backgrounds in performance mode
3. Gold must appear somewhere on every host
4. Neon orange, cyan, or magenta as primary accent family
5. No mismatch between host lighting and room lighting
