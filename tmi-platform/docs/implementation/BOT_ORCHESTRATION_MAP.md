# Bot Orchestration Map

## Core bots for launch
- Discovery Fairness Bot: maintains inverted exposure order.
- Sequence Bot: enforces Artist -> Article -> Random rhythm.
- Random Page Balancer Bot: controls mix and cooldown.
- Momentum Bot: computes movement signals.
- Originality Bot: boosts unique creators.
- Sponsor Match Bot: matches brand-safe placements.
- Earnings Bot: computes sponsor rewards and point grants.
- Nav Index Bot: updates search index and jump directory.

## Bot chain examples
### Discovery chain
signals -> fairness scoring -> exposure list -> preview wall placement

### Monetization chain
sponsor intake -> approval -> placement -> interaction tracking -> payout distribution

### Magazine chain
issue plan -> sequence build -> random page balancing -> publish -> analytics feedback

## Safety controls
- Anti-manipulation checks before rank movement.
- Rate limits on engagement-derived boosts.
- Rollback plan for mis-ranked issue builds.
