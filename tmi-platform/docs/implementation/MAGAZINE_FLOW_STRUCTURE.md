# Magazine Flow Structure

## Canonical sequence
`Artist page -> Article/News page -> Random page -> repeat`

## Random page pool
- Sponsor spread
- Billboard snippet
- Poll results
- Trivia/game teaser
- Contest card
- Horoscope/lifestyle insert
- Throwback card
- Seasonal card

## Ranking model
- Exposure order is inverted to promote low-exposure artists first.
- Prestige rank is independent and shown as a badge.

## Reading memory
- Keep last 20 visited pages in session memory.
- Allow bookmarking for persistent revisit.

## Navigation obligations
- Fast jump bar on every major page.
- Global command palette search for sections and entities.
- Recent rail visible for backtracking random pages.
