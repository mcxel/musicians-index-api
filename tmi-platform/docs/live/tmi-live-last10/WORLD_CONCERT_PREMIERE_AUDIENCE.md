# WORLD_CONCERT_SYSTEM.md
## World Concerts — Global Scale Live Events in Venue Environments

---

## CONCEPT
A World Concert is a major platform-wide event.
Multiple venues run simultaneously — globally distributed.
Every audience member in every venue watches the same performance.
Artists perform on the World Stage.

---

## WORLD CONCERT SCALE

```
ARTIST on WORLD STAGE
         ↓
    LIVE BROADCAST SIGNAL
         ↓
┌────────────────────────────────────────────────┐
│  Venue 1        Venue 2        Venue 3          │
│  (Arena)        (Club)         (Amphitheater)   │
│  25,000 fans    500 fans       5,000 fans       │
└────────────────────────────────────────────────┘
```

**All venues see the same performance.**
**Each venue feels like their own experience.**

---

## WORLD STAGE

The World Stage is a special venue tier:
- Only unlocked for major events or Diamond-tier artists
- No seat limit
- Simultaneous multi-venue broadcast
- Full cinematic production quality
- Real-time global viewer count displayed in all venues
- Crowd reactions pooled from all venues globally

---

## WORLD CONCERT VENUE ATMOSPHERE

Each venue gets its own version of the concert experience:

| Venue | World Concert Feel |
|---|---|
| Basement | Intimate broadcast, fans feel "close" |
| Club | Dance energy, crowd all up |
| Amphitheater | Premium outdoor concert feel |
| Arena | Full stadium production |
| World Stage (primary) | No cap, global audience display |

---

## WORLD CONCERT FEATURES

- **Global crowd meter** — shows total viewers across all venues
- **City display** — shows which cities/venues are watching
- **Synchronized light shows** — all venues sync their lighting
- **Shared reaction explosions** — if 10,000 people react at once, every venue feels it
- **Artist can address specific venues** — "Detroit, are you in there?"
- **Simultaneous tip drops** — tips aggregate across all venues
- **Real-time chart impact** — World Concert performance boosts artist's index rank

---

## WORLD CONCERT PRODUCTION MODES

| Mode | Description |
|---|---|
| SCREEN_CONCERT | Artist on main screen only |
| AVATAR_CONCERT | Artist avatar performs on each venue's stage |
| HYBRID_CONCERT | Screen feed + avatar reactions |
| INTERACTIVE_CONCERT | Artist chooses which venue gets spotlight moments |

# PREMIERE_SYSTEM.md
## Premieres — Red Carpet + Watch Party Events in Venues

---

## CONCEPT
A Premiere is a special event for releasing music, videos, albums, or show debuts.
The audience gathers in a venue to watch the premiere together.
It feels like a movie premiere or album release party.

---

## PREMIERE TYPES

| Type | What Premieres | Venue Style |
|---|---|---|
| Music Video Premiere | New music video drops | Premiere Theater |
| Album Release Premiere | New album listening party | Concert Hall, VIP |
| Show Debut Premiere | First episode of a new show | Theater-style venue |
| Battle Premiere | Major battle reveal | Arena or Battle Stage |
| Artist Introduction | New artist announcement | Special reveal venue |
| Award Premiere | Award show broadcast | Prestige Hall |

---

## PREMIERE VENUE — THE PREMIERE THEATER

A dedicated venue for premiere events:
- Red carpet entry zone
- Theater-style seating (angled, tiered)
- Giant main screen (premium resolution)
- Side screens for reactions
- VIP lounge boxes
- Sponsor wall surfaces
- Julius in red-carpet mode
- Special lighting (premiere glow)

---

## PREMIERE FLOW

```
1. RED CARPET PHASE (pre-premiere)
   - Attendees arrive
   - Julius plays host
   - Countdown timer visible
   - Sponsor displays active
   - Crowd chatter/hype builds

2. OPENING SEQUENCE
   - Curtain drops
   - Premiere lighting activates
   - Host MC (optional) opens
   - Special intro plays

3. PREMIERE MOMENT
   - Main screen activates
   - Content plays (video/album/show)
   - Audience reacts in real-time
   - Reactions visible to all

4. REVEAL / RELEASE
   - Content goes live to full platform
   - Artist may appear on screen/stage
   - Q&A window (optional)
   - Crowd celebration FX

5. AFTER-PARTY
   - Lights shift to party mode
   - Music plays
   - Social sharing enabled
   - Artist available for chat
```

---

## PREMIERE EXCLUSIVITY TIERS

| Tier | Access |
|---|---|
| Public Premiere | Open to all users |
| Invite Premiere | Artist selects invitees |
| VIP Premiere | Gold+ tier only |
| Private Premiere | Closed list only |

# AUDIENCE_EXPERIENCE_ENGINE.md
## How Audiences Experience Live Events in Venues

---

## PURPOSE
Defines the complete audience experience — what fans see, feel, and do when they're sitting in a venue watching any live event.

---

## AUDIENCE VIEW

### What the Audience Sees From Their Seat

**Front Row:**
- Close-up of performer/screen
- Can see artist's face, expressions, details
- Most immersive
- Crowd behind them visible

**Mid Row:**
- Balanced view of stage AND crowd
- Can see reactions around them
- Best "full experience" view

**Back Row:**
- Full venue overview
- Stage appears smaller but complete
- Can see all crowd reactions
- Feels like a real venue back row

**VIP Box:**
- Side angle, elevated
- Private feel
- Sees both stage and crowd at 45-degree angle
- Premium visuals

**Balcony:**
- Bird's-eye view
- Full room visible
- Cinematic perspective

---

## AUDIENCE INTERACTION CONTROLS

While seated in a live event, audience can:

| Action | How | Effect |
|---|---|---|
| React | Emoji tap | Sends emoji animation across screen |
| Cheer | Hold cheer | Adds to crowd energy meter |
| Tip | Tip button | Money/points sent to artist |
| Vote | Julius poll board | Real-time vote |
| Chat | Message box | Moderated chat feed |
| Clap | Clap button | Audio clap added to crowd sound |
| Stand up | Stand action | Avatar stands, visible to nearby seats |
| Bookmark | Save button | Saves this moment |
| Share | Share button | Creates clip for social |
| Flashlight | Light button | Phone light effect (concert feel) |

---

## CROWD ENERGY METER

The crowd energy meter is always visible during live events:

- Fills as audience reacts
- Peaks trigger special effects in venue
- Host or artist can see their crowd meter
- Artist can pump the crowd intentionally
- Low energy → Julius tries to hype the crowd
- Peak energy → special FX: confetti, light explosion, sound burst

---

## AUDIENCE AVATARS IN VENUE

Each audience member appears as an avatar in their seat:

**Default appearance:**
- Basic avatar that matches user's profile
- Generic if no profile set

**Active appearance:**
- Animated when reacting
- Stands when standing
- Arms raise during cheers
- Flashlight effect visible

**Proximity visibility:**
- Nearby seats see detailed avatars
- Far seats see crowd-density waves
- Performance mode: simplified crowd for low-power

---

## SOUND EXPERIENCE

The audience hears:

| Sound Layer | What It Is |
|---|---|
| `artist_audio` | Artist performance (main) |
| `crowd_ambient` | Crowd around you cheering, talking |
| `venue_acoustic` | Room reverb + acoustic profile per venue |
| `event_sfx` | Special effects tied to moments |
| `julius_audio` | Julius nearby sounds |
| `reaction_audio` | When you react, you hear it |

**Spatial audio (future):** sounds come from their direction in the venue.

---

## SEAT UPGRADE SYSTEM

During a live event, audience members can upgrade their seat:

| Upgrade | Cost | Benefit |
|---|---|---|
| Move to front rows | Points | Better close-up view |
| VIP Box | Points or subscription | Private angle + exclusive chat |
| Backstage Pass | Premium | See backstage view + artist prep area |
| Artist Eye | Special | See from artist's perspective |

---

## LATECOMERS

If a viewer arrives after the event starts:
- Fade-in entry to their assigned seat
- Catch-up summary shows what they missed
- Full experience from that point forward
- Cannot scroll back (live is live)

---

## POST-EVENT

After the event ends:
- Outro sequence plays
- Host wraps up (if applicable)
- Audience release animation (crowd filters out)
- Replay available based on artist settings
- Artist rank/momentum updated
- Audience points earned
- Venue transitions to idle state
