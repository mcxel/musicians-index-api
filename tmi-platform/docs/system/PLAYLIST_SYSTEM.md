### BerntoutGlobal XXL / The Musician's Index

---

## PLAYLIST TYPES

| Type | Creator | Visibility | Notes |
|---|---|---|---|
| Artist Playlist | Any artist | Public | Artist's official track collections |
| Fan Playlist | Any user | Public or Private | Fan-curated via external links |
| Platform Playlist | Editorial team | Public | Curated by TMI editors |
| Genre Radio | System (auto) | Public | Auto-generated by genre-radio-bot |
| Fan Club Playlist | Artist (fan club feature) | Fan Club Members | Exclusive to tier |

---

## PLAYLIST ITEMS

A playlist item can be:
- External song link (Spotify, Apple Music, SoundCloud, YouTube)
- Beat from the TMI Beat Marketplace
- A TMI artist's uploaded track

TMI does not host full tracks for streaming in Phase 1. Links to external platforms are embedded.

---

## GENRE RADIO

Automatically generated playlists per genre:
- genre-radio-bot runs nightly
- Selects 20 tracks from discovery-first artists in that genre (low-viewer, active)
- Playlist rotates weekly
- Users follow a genre → genre radio auto-added to their feed

## MODELS (Prisma)

```prisma
model Playlist {
  id          String         @id @default(cuid())
  creatorId   String
  title       String
  description String?
  type        String         // 'artist' | 'fan' | 'platform' | 'genre_radio'
  isPublic    Boolean        @default(true)
  coverUrl    String?
  items       PlaylistItem[]
  followers   Int            @default(0)
  createdAt   DateTime       @default(now())
}

model PlaylistItem {
  id           String   @id @default(cuid())
  playlistId   String
  playlist     Playlist @relation(fields: [playlistId], references: [id], onDelete: Cascade)
  type         String   // 'external_link' | 'beat' | 'upload'
  externalUrl  String?
  beatId       String?
  title        String
  artistName   String
  position     Int
  addedAt      DateTime @default(now())
}
```
