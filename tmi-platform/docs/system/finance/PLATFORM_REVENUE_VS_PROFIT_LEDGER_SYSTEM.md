## LEDGER ENTRY TYPES

Every money movement creates a Transaction record:

```
type: 'tip_gross'          — full tip amount received
type: 'tip_artist_share'   — 70% to artist wallet
type: 'tip_platform_fee'   — 30% platform kept
type: 'stripe_fee'         — Stripe processing cost (debit)
type: 'beat_sale_gross'    — full beat license amount
type: 'beat_artist_share'  — 80% to producer wallet
type: 'beat_platform_fee'  — 20% platform kept
type: 'ticket_gross'       — full ticket amount
type: 'ticket_artist_share'— 75% to event host wallet
type: 'ticket_platform_fee'— 25% platform kept
type: 'fan_club_gross'     — full subscription amount
type: 'fan_club_artist_share' — 70% to artist
type: 'fan_club_platform_fee' — 30% platform
type: 'refund'             — customer refund (negative)
type: 'chargeback'         — chargeback loss (negative)
type: 'operating_cost'     — hosting, APIs, etc. (debit from platform)
type: 'reserve_allocation' — moved to reserve bucket (debit from distributable)
type: 'owner_distribution' — moved to owner (debit from distributable)
type: 'credit_purchase'    — fan credits bought (100% platform)
type: 'avatar_purchase'    — avatar store (100% platform)
type: 'artist_payout'      — artist withdrawal to Stripe Connect (debit)
type: 'owner_payout'       — owner distribution to PayPal/bank (debit)
```
