## WHAT EACH ROOM CONTAINS

Every room pack includes:
1. ROOM_IDENTITY — Name, tier, concept, feel, colors
2. ROOM_LAYOUT — Floor plan, dimensions, zones
3. ROOM_SEATING_MAP — Seat types, rows, capacity
4. ROOM_SIGHTLINES — View rules from every position
5. ROOM_STAGE_AND_SCREEN — Stage surface + screen types
6. ROOM_LIGHTING_AND_CURTAINS — Light profiles + event triggers
7. ROOM_AUDIO_PROFILE — Acoustic character + sound layers
8. ROOM_HOST_BLOCKING — Where hosts, Julius, VEX can be
9. ROOM_CROWD_BEHAVIOR — Crowd states, energy cap
10. ROOM_CAMERA_MAP — All available camera views
11. ROOM_ANIMATION_RULES — What animates + event triggers
12. ROOM_EVOLUTION_PATH — How room transforms to next tier
13. ROOM_PERFORMANCE_BUDGET — Resource limits
14. ROOM_PROOF_CHECKLIST — Completion verification
